# Emerging Technologies — 新兴科技全景

[![Live Demo](https://img.shields.io/badge/Live-emerging--tech--nine.vercel.app-00e4a0)](https://emerging-tech-nine.vercel.app)

A bilingual (Chinese–English) panoramic guide to emerging technologies across six fundamental domains converging toward a singular technological future.

## Sections

- **Five Technological Frontiers** — Energy, Information, Space-Time, Matter, Biology with goals, key technologies, and color-coded domain cards
- **Evolution Timelines** — How each domain evolved (Language → GPT, Stone → Nano, Fire → Nuclear, etc.) plus Technology → Core Products mapping
- **Engine Evolution** — Visual progression from Rocket → Jet → Ion → Fusion → Warp Drive
- **Grand Convergence 2046** — Four predicted convergences: Capitalism/Socialism, AI/Human consciousness, Quantum/Relativity, Digital/Physical worlds
- **Complete Overview Table** — All six domains at a glance with focus, technologies, goals, and evolution paths

## Design

Dark sci-fi aesthetic with six-color domain coding (amber/energy, blue/information, purple/space-time, green/matter, pink/biology, gold/consciousness). Multi-colored particle network hero canvas, Outfit + Noto Serif SC + IBM Plex Mono typography, sticky navigation, scroll-reveal animations.

## Tech

Single-page HTML/CSS/JS — no build step, no dependencies.
